Simulação Black Hole

Experimento visual em Canvas simulando partículas orbitando um buraco negro para estudar física básica e animações.

HTML5 Canvas JavaScript Simulação

Sobre o Projeto

Este projeto surgiu como um estudo de física básica aplicada à computação gráfica. A ideia é representar um buraco negro no centro da tela e múltiplas partículas orbitando ao redor, sofrendo influência de um campo de gravidade simplificado.

Ele foi desenvolvido ao longo do meu primeiro ano de experiência em programação, e ainda é um projeto ativo que uso para experimentar novas ideias de visualização.

Objetivos Técnicos

  • Modelar um campo de força simples para atrair partículas para o centro.
  • Trabalhar com atualizações em tempo real usando JavaScript e requestAnimationFrame.
  • Explorar composição de cores, gradientes e efeitos de trilha no Canvas.
  • Garantir boa performance mesmo com muitas partículas em tela.

Conceitos Estudados

Matemática

  • Vetores para posição, velocidade e aceleração.
  • Cálculo de distância entre pontos.
  • Força inversamente proporcional à distância.

Renderização

  • Desenho de milhares de partículas no Canvas.
  • Efeito de trilha usando transparência no fundo.
  • Camadas de brilho para o núcleo do buraco negro.

Arquitetura de Código

  • Separação entre lógica de física e desenho.
  • Uso de funções puras para atualização de estado.
  • Organização em pequenos módulos reutilizáveis.

Aprendizados

Este projeto me ajudou a ganhar confiança em manipular o Canvas, lidar com animações complexas e pensar em termos de simulação. Também foi uma forma prática de treinar estruturas de dados simples e otimização em JavaScript.

Hoje eu o utilizo como laboratório: sempre que quero testar uma ideia nova de visual ou de física, costumo começar experimentando aqui.
