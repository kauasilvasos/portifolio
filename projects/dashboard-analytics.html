Sobre o Projeto

Dashboard analytics moderno com visualização de dados em tempo real. Desenvolvido com TypeScript para maior robustez, utiliza Chart.js para gráficos interativos e WebSocket para atualizações em tempo real. Totalmente responsivo e com suporte a múltiplos temas.

Funcionalidades

  • Gráficos interativos em tempo real
  • Múltiplos tipos de visualizações
  • Filtros avançados de dados
  • Exportação de relatórios
  • Temas claro e escuro
  • Responsivo em todas as telas
  • Sincronização com backend via WebSocket
  • Cache inteligente de dados
